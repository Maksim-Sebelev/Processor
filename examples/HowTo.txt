(!!!) firstly change your directory to "examples"


1. To translate first example file (exaple1.asm) to machine code and run it use:
./build/processor -compile example1/example1.asm example1/example1.bin -run example1/example1.bin


second example:
./build/processor -compile example2/example2.asm example2/example2.bin -run example2/example2.bin

third example:
./build/processor -compile example3/example3.asm example3/example3.bin -run example3/example3.bin



2. if you want only translate .asm file to machine code use:

first example:
./build/processor -compile example1/example1.asm example1/example1.bin

second example:
./build/processor -compile example2/example2.asm example2/example2.bin

third example:
./build/processor -compile example3/example3.asm example3/example3.bin 



3. if you want only run machine code use:

first example:
./build/processor -run example1/example1.bin


second example:
./build/processor -run example2/example2.bin

third exaple:
./build/processor -run example3/example3.bin
